# PopMovie

### API를 활용한 현재 상영중인 영화 & 인기영화 그리고 TVshow를 확인할 수 있는 웹서비스
https://jahokoo.github.io/ReactJS/
## 개발환경
React<br>
React-Router 6v<br>
SCSS<br>
Swiper<br>
TMDB API<br>
